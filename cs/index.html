Dobrovolník

Pyvec nikomu neříká, co má dělat. Pomáhá těm, kteří se o něco snaží — je to servisní organizace. Založení srazu ve tvém městě? Kurz na tvé škole? Workshop? Screencast? Pojďmě spolu vymyslet, jak to zrealizovat! Ozvi se nám na info@pyvec.org.

  • Konference PyCon CZ

    cz.pycon.org

    Každoroční největší setkání lidí se zájmem o jazyk Python v ČR. Je součástí mezinárodní sítě podobných konferencí.

    Jak pomáháme: Proplácení vybavení, účetnictví, smlouvy, právní servis

  • Nauč se Python

    naucse.python.cz

    Otevřené a volně dostupné materiály pro začátečnické i pokročilé kurzy jazyka Python. Jediné svého druhu v češtině, neustále udržované a rozšiřované.

    Jak pomáháme: Provoz

  • PyLadies CZ

    pyladies.cz

    Mezinárodní aktivita, která se snaží s pomocí jazyka Python sblížit ženy a IT. České skupiny pořádají několikaměsíční bezplatné začátečnické kurzy.

    Jak pomáháme: Proplácení vybavení, smlouvy, propojení místních skupin

  • Srazy Pyvo

    pyvo.cz

    Neformální programátorské srazy zaměřené na odborné přednášky, networking nebo vzdělávání, konané pravidelně každý měsíc.

    Jak pomáháme: Proplácení vybavení, propojení místních skupin

  • PyWorking

    pyworking.cz

    Prakticky zaměřené srazy pro pokročilé začátečníky, kde si lze cokoliv vyzkoušet nebo konzultovat. V Praze navíc doplněné o občasné celodenní workshopy.

    Jak pomáháme: Proplácení vybavení, smlouvy

  • Django Girls

    djangogirls.org/events

    Jednodenní workshopy na Python & Django, pod mezinárodní hlavičkou, jejichž snahou je usnadnit ženám první kontakt s technologiemi.

    Jak pomáháme: Proplácení vybavení, smlouvy, propojení místních skupin

  • python.cz

    python.cz

    Hlavní zdroj informací a rozcestník pro českou Python komunitu.

    Jak pomáháme: Provoz

  • Komunitní blog

    blog.python.cz

    Zprávy ze světa české Python komunity. Kdokoliv může přispět svým článkem.

    Jak pomáháme: Provoz

  • Komunitní dokumentace

    docs.pyvec.org

    Dokumentace fungování komunity, která shrnuje i know-how pro organizátory akcí, jako jsou srazy, workshopy, kurzy, apod.

    Jak pomáháme: Provoz
